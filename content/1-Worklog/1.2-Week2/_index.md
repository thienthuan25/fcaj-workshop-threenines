### Week 2 Goals:

* Understand the core concepts of High Availability, Scalability, and Elasticity. Master and practice with Elastic Load Balancing (ELB) and Auto Scaling Groups (ASG).
* Deep dive into Amazon S3: data management, security (Bucket Policy), Static Website Hosting, Versioning, Replication, and Storage Classes.
* Grasp large-scale data migration methods through the AWS Snow Family and Storage Gateway.
* Understand and practice deploying relational database services (RDS, Aurora) and NoSQL (DynamoDB, ElastiCache). Gain an overview of Data Analytics services on AWS.
* Get familiar with the diverse compute ecosystem beyond EC2: Containers (Docker, ECS, EKS), Serverless architecture (AWS Lambda, API Gateway), and AWS Lightsail.

### Week 2 Achievements:

* Clearly understood and distinguished important architectural concepts: High Availability, Scalability, and Elasticity.
* Successfully configured and deployed an Application Load Balancer (ALB) to distribute traffic.
* Set up Auto Scaling Groups (ASG) combined with auto-scaling strategies to ensure application availability.
* Mastered Amazon S3 features: 
  * Created buckets and configured Bucket Policies for data security.
  * Set up S3 for Static Website Hosting.
  * Enabled and managed S3 Versioning; configured S3 Replication.
  * Mastered S3 Storage Classes, S3 Express One Zone, and Encryption mechanisms.
* Grasped hybrid data migration/storage solutions with AWS Snow Family and AWS Storage Gateway; clearly understood Snowball Edge pricing.
* Successfully deployed and practiced basic database administration on AWS:
  * Successfully initialized Amazon RDS and understood deployment options.
  * Practiced with Amazon DynamoDB and configured Global Tables.
* Gained a comprehensive overview of the Database and Data Analytics ecosystem on AWS, distinguishing the use-cases for: ElastiCache, Redshift, EMR, Athena, QuickSight, DocumentDB, Neptune, Timestream, Managed Blockchain, Glue, and DMS.
* Understood the fundamentals of Containers (Docker) and AWS container management services (ECS, Fargate, ECR, EKS).
* Grasped the Serverless computing model, successfully practicing creating and running AWS Lambda functions.
* Understood the overview of Amazon API Gateway and AWS Batch, and practiced quickly provisioning virtual servers with Amazon Lightsail.
* Passed the multiple-choice review quizzes assessing competency in elastic computing (ELB & ASG) and storage (S3) topics.
